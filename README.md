# 🌱 Smart Agri Doctor

Smart Agri Doctor is an AI-powered crop disease detection web application that analyzes plant leaf images and provides disease diagnosis, treatment recommendations, and prevention tips. The system also supports multilingual responses, voice explanations, and weather-based disease risk alerts to assist farmers and agricultural researchers.

## 📌 Features

- 🌿 **Crop Disease Detection**  
  Upload or capture a plant leaf image to detect possible crop diseases.

- 🧠 **AI-based Diagnosis**  
  Uses a multimodal AI model to analyze plant leaf images.

- 💊 **Treatment Recommendations**  
  Provides solutions and prevention methods for detected diseases.

- 🌍 **Multilingual Support**  
  Supports English, Hindi, and Telugu.

- 🔊 **Voice Explanation**  
  Converts diagnosis results into speech for accessibility.

- 🌦 **Weather-based Disease Risk**  
  Shows potential disease risk based on temperature and humidity.

- 🔎 **Disease Information Search**  
  Users can search for information about specific plant diseases.

- 📥 **Download Report**  
  Allows users to download the diagnosis report.

## 🛠 Technologies Used

- **Python**
- **Streamlit**
- **Groq API (LLM Vision Model)**
- **Pillow** – Image Processing
- **gTTS** – Text-to-Speech
- **Requests** – Weather API
